# Ficha Editável - Mage: The Ascension

Projeto HTML + CSS + JavaScript nativo, pronto para rodar no GitHub Pages.

## Como usar

1. Abra `index.html` localmente ou publique a pasta no GitHub Pages.
2. Preencha os campos da ficha.
3. Clique em **Salvar JSON** para baixar o estado da ficha.
4. Use **Carregar JSON** para restaurar uma ficha salva.

Não usa frameworks, build ou dependências externas.
